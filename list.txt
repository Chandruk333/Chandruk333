Creating a list in Python

# Python program to demonstrate
# Creation of List
 
# Creating a List
List = []
print("Blank List: ")
print(List)
 
# Creating a List of numbers
List = [10, 20, 14]
print("\nList of numbers: ")
print(List)
 
# Creating a List of strings and accessing
# using index
List = ["books", "For", "pens"]
print("\nList Items: ")
print(List[0])
print(List[2])


output:
Blank List: 
[]

List of numbers: 
[10, 20, 14]

List Items: 
books
pens
